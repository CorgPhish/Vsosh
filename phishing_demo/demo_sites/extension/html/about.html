CorgPhish

Версия 1.0.0

CorgPhish - это расширение для браузера, предназначенное для защиты от фишинговых сайтов с особым фокусом на русскоязычный сегмент интернета.

В отличие от стандартных решений, CorgPhish использует продвинутые алгоритмы машинного обучения, которые работают полностью на вашем устройстве, не передавая данные на внешние серверы.

Основные возможности

  • Локальная работа - Все вычисления производятся прямо в браузере, без отправки URL на сервер
  • Интеграция с Google Safe Browsing - Дополнительная проверка через базу данных Google
  • Машинное обучение - Использование XGBoost модели для определения фишинговых сайтов
  • Поддержка кириллицы - Специальные алгоритмы для обнаружения фишинга на русскоязычных сайтах
  • Анализ доменов - Определение сходства с популярными сервисами для выявления подделок
  • Подробные предупреждения - Информативные уведомления с объяснением причин блокировки

Как это работает?

CorgPhish анализирует URL-адреса и содержимое страниц, которые вы посещаете, используя сочетание следующих технологий:

  1. Проверка домена через списки известных фишинговых и безопасных сайтов
  2. Анализ URL с помощью модели машинного обучения XGBoost
  3. Проверка сходства доменов с популярными сервисами
  4. Интеграция с Google Safe Browsing API для дополнительной верификации

Благодаря использованию локальной модели машинного обучения, расширение работает быстро и не передает информацию о посещаемых вами сайтах на сторонние серверы, обеспечивая максимальную приватность.

Настройки

CorgPhish предлагает гибкие настройки, позволяющие:

  • Включать/выключать различные методы проверки
  • Настраивать интеграцию с Google Safe Browsing API
  • Управлять уведомлениями и предупреждениями
  • Создавать собственный белый список доверенных сайтов

Для доступа к настройкам нажмите правой кнопкой мыши на иконку расширения и выберите "Настройки".
